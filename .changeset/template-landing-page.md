Scaffold a landing page. The site root was a redirect into `/docs/introduction`; it is now a real `+page.svelte` built from the new landing sections, with a hero carrying your site title, a six-cell feature grid, and a closing call to action. So `npm create svelte-docsmith` produces a complete site rather than only a docs section, and the page doubles as a worked example to edit. Delete it and add a `+page.ts` redirect back if you'd rather not have one.
