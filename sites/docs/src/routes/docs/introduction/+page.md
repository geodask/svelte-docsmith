## What is Svelte DocSmith?

Svelte DocSmith is a framework for building documentation sites with Svelte 5
and SvelteKit. Your interactive examples live inside one real, stateful app —
not sandboxed as isolated islands.

## Why another docs tool?

Most Svelte docs are written by hand or bolted onto a general-purpose static
site generator. DocSmith gives you the markdown pipeline, the layout, and the
navigation out of the box, so you can focus on the content.

### Highlights

- Markdown compiled to real Svelte routes via mdsvex.
- Syntax highlighting powered by Shiki, running on the HAST tree.
- Sidebar navigation derived from your content — never hand-written.
